#  Keras Tuner IMDB Classification

##  Overview

This project builds a **sentiment classification model** using the IMDB dataset and applies **Keras Tuner** to find the best hyperparameters automatically.

---

##  Dataset

* IMDB Movie Reviews (from TensorFlow)
* Binary classification:

  * 0 → Negative
  * 1 → Positive

---

##  Approach

* Used **Embedding + Dense Neural Network**
* Applied **Keras Tuner (Random Search)** to tune:

  * Embedding size
  * Number of units
  * Learning rate

---

##  Technologies

* TensorFlow / Keras
* Keras Tuner
* Python

---

##  Key Learning

* Hyperparameter tuning improves model performance
* Automation is better than manual trial
